## Project Overview — D-Itinerary Website

D-Itinerary is a travel assistance website that provides:

* A visually appealing Home Page
* Hotels Page with destination images
* Restaurants Page with testimonials
* Search Page for enquiries
* Terms & Conditions Page
* Login Page
The design prioritizes clean layout, good typography, responsive sections, and user-friendly navigation. 

## Fundamental Components

* Fixed Navigation Bar
* Hero Image Section
